Projeto - Patas do Bem

Projeto web de uma ONG fictícia, com páginas de Início, Projetos e Cadastro de voluntários, desenvolvido com HTML, CSS e JavaScript.

 Objetivo do Projeto

Desenvolver uma aplicação web dinâmica e acessível, aplicando boas práticas de:

HTML semântico e CSS moderno

Layout responsivo com Grid e Flexbox

Componentes interativos (cards, formulários, botões)

Validação de dados com JavaScript

Controle de versão profissional com Git/GitHub

Deploy otimizado para produção

Tecnologias Utilizadas

HTML5

CSS3 

JavaScript

Git & GitHub

Live Server (para testes locais)
